# Lab Report 3: Researching ``find`` Command
## 1. ```find [path] -type [file type]```
This command outputs the files that are of the specificed file type (shown below) found from within the given path. 

File Type Options:

             b       block special
             c       character special
             d       directory
             f       regular file
             l       symbolic link
             p       FIFO
             s       socket
             
### Example 1:
***
#### Input:
```
$ find technical/ -type d 
```
#### Output:
